method eyes::open()

This method is called once, at the start of an session.

Applicability

Languages : Java, also available in

Platform : Selenium, also available on

Syntax

WebDriver open(WebDriver driver, String appName, String testName,RectangleSize viewportSize)

WebDriver open(WebDriver driver, String appName, String testName)

Description

This method is called once, at the start of the test and before any calls to check*. or test* The appName and testName parameters are two of the multiple factors that govern the baseline for these tests. These strings appear in the various screens.

Notes

A version of this API exists that does not take a viewportSize parameter. In this case the default is the viewport size is taken as the viewport size of the window in which the code is executing. The viewport size may be set at a later point using the API setViewportSize(RectangleSize size)
